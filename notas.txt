En el terminal se utiliza "pip install flask" para flask

                          "pip install pymongo" para los paquetes de mongo

                          "pip install certifi" para conectarse a mongo remotamente

                          "pip install pymongo[srv]" para datos MS que no dejara conectarnos si no esta instalado

app.py es la vista principal
database.py es el archivo para conectarse a la BD
product.py es la clase producto a utilizar
